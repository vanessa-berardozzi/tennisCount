## Documentation

The project uses TypeDoc for documentation generation. To view the documentation:

1. Generate the docs:
```bash
npm run docs
```

2. Open `docs/index.html` in your browser

## Technical Stack

- Next.js 15
- TypeScript
- Jest for testing
- React Testing Library

## Installation

```bash

npm run dev
```
